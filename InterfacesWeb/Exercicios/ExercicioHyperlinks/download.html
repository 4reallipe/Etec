Download de arquivo

Este exemplo mostra um hiperlink que permite baixar um arquivo local (por exemplo um PDF).

O arquivo guia.pdf deve estar na mesma pasta que este arquivo.

Use o atributo download para sugerir o nome do arquivo salvo.

Baixe o guia: Download do Guia

Observação: na prática, o arquivo real pode ser PDF, ZIP, etc. Aqui você pode usar um arquivo PDF real ou um arquivo .txt renomeado para .pdf.
